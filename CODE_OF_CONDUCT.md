# Code of Conduct

## Our Standard

This project expects respectful, technical, and constructive collaboration.

Examples of expected behavior:

- Discuss design tradeoffs with evidence
- Report bugs with reproducible context
- Keep reviews focused on code and behavior

Unacceptable behavior includes:

- Personal attacks
- Harassment
- Bad-faith participation
- Publishing private user data captured through accessibility output

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions
that violate this code of conduct.
