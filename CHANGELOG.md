# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog, and this project follows Semantic Versioning.

## [0.1.0] - 2026-04-10

### Added

- Multi-module Gradle project for `core`, `android`, and `sample`
- Accessibility-based top window dump pipeline
- JSON export model and sample JSON output
- Sample app with dump, save, and share actions
- GitHub Actions build and release workflows
- Maven publishing configuration for `jar` and `aar`
